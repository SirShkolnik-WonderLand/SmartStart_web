- **HTTPS Only** (HSTS enabled)
- **Strict Transport Security** (max-age: 1 year)

- **Content Security Policy** (CSP)
- **X-Frame-Options** (Clickjacking protection)
- **X-Content-Type-Options** (MIME-sniffing protection)
